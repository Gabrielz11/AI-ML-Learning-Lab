## 🚀 Key Features / Funcionalidades Principais

- **Dual Modules:** Classification (Breast Cancer) and Regression (Diabetes).
- **Explanation Engine:** Translates metrics into analogies (e.g., "Decision Tree = Guess Who?").
- **Interactive Simulator:** Test the trained models with your own custom inputs.
- **Bilingual Interface:** Toggle between Portuguese and English on the fly.
- **XAI (Explainable AI):** Feature importance visualization to see what the model "thinks" is important.

---

## 🛠️ Tech Stack / Tecnologias

- **Language:** Python
- **Interface:** Streamlit
- **ML Engine:** Scikit-learn
- **Data Viz:** Seaborn & Matplotlib
- **Data Handling:** Pandas & Numpy


2.  **Run the app / Execute o app:**
